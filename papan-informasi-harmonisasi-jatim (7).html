📋 Info Terkini

📤 Update Data Harmonisasi

Upload file Excel jadwal rapat harmonisasi terbaru. Sistem akan otomatis membaca semua sheet, menghitung jumlah rancangan per daerah, status (Diterima / Dikembalikan / Penjadwalan Ulang), dan jenis (Raperda / Raperkada) — lalu memperbarui peta dan statistik di Papan Informasi.
📊
Klik atau seret file Excel ke sini
Format: .xlsx — sesuai struktur "Jadwal Rapat Harmonisasi"
💾 Memuat informasi penyimpanan...

📋 Cara Kerja

1
Siapkan file Excel dengan kolom Pemrakarsa, STATUS, dan Jenis (Raperda/Raperkada) di setiap sheet — sama seperti format jadwal rapat harmonisasi yang sudah ada.
2
Upload file melalui kotak di atas. Sistem otomatis mendeteksi header tabel di tiap sheet, lalu mencocokkan nama daerah dari kolom Pemrakarsa.
3
Periksa tabel pratinjau yang muncul — pastikan jumlah per daerah sudah sesuai sebelum disimpan.
4
Klik Simpan & Tampilkan. Peta, statistik, dan ticker di tab "Papan Informasi" akan otomatis ter-update.
5
Status yang dikenali: Diterima, Dikembalikan, Penjadwalan Ulang. Jenis yang dikenali: Raperda dan Raperkada (termasuk Raperbup/Raperwal/Rapergub).

🗑️ Reset Data

Kembalikan papan informasi ke data contoh awal (data Excel Jadwal Harmonisasi 2026).
